## 鸥的宝藏

​	$Input$ $file:$ 			  $standard$ $input$

​	$Output$ $file:$			 $standard$ $output$

​	$Time$ $limit:$			  $1$ $second$

​	$Memory$ $limit:$		$256$ $megabytes$

「冒险！一起去寻找小岛上的秘密吧！」

在小学的阁楼里，鸥找到了一本陈旧的笔记本和一个被锁上的宝箱。

笔记本的扉页上，写着一行字：

> **「十年后为了能让我们再聚到一起，我们将今天找到的这个岛的秘密封印在这个宝箱里。」**

笔记本的末页隐约记载着一段往事：

十年前，伙伴们各自挑选了若干个**正整数**并记录在册。这些数字都非常小（均小于 $16$）。

笔记本上写着：

> 「当这 $n$ 个数字的 **按位或（bitwise OR）** 运算结果恰好等于宝箱上刻着的神秘数字 $x$ 时，宝箱将会开启。」

时光荏苒，笔记本上的 $n$ 个数字早已模糊不清。但鸥记得，当时参与的四位小伙伴挑选数字时非常随性，可以看作是在 $[0, 15]$ 范围内**独立均匀地随机**挑选。

给定 $n$ 和 $x$，请你判断：宝箱**是否确定**可以被打开？

### Input

第一行包含一个整数 $T$ ($1 \le T \le 10^5$)，表示测试用例的数量。

每个测试用例包含一行，有两个整数 $n$ 和 $x$ ($10^4 \le n \le 10^{18}$，$0 \le x \le 10^{18}$)，分别表示笔记本上记录的数字个数和宝箱上的数字。

### Output

对于每个测试用例，如果宝箱确定能打开，输出 `YES`；否则输出 `NO`。

### Example

#### standard input

```
1
10000 16
```

####  standard output

```
NO
```

### Note

对于第一个测试样例，所有选取的数都小于 $16$，它们的运算结果不可能是 $16$ 。
