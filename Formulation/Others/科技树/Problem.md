# U659819 科技树

时间限制：1 s

内存限制：512 MB

## 题目描述

最近终末地十分火爆，游戏中玩家需要在塔卫二上发展科技实现自动化。为了防止玩家越科技，设计师给每项科技都设置了前置科技，换而言之，你不能跳科技。但是 CubeCat 在游玩过程中觉得科技树并不完美，所以他重新设计了一个可能更合理的科技树。

具体来说，CubeCat 的科技树一共包含 $n$ 个节点和 $m$ 个关系，每个关系给出的格式为 `u v` ，表示科技点 $u$ 是 $v$ 的前置。CubeCat 设计好这个新的科技树后就睡觉了，但是第二天起床后他发现他的设计被破坏了，部分科技和它前置之间的关系被删除了。CubeCat 在睡了一觉后忘记了之前怎么设计的，只记得哪些关系被删除了，但是忘记了谁是谁的前置。这可不太好！要是随意设置方向可能会导致部分科技永远无法解锁，于是他把问题交给了你，请你告诉他那些未知的关系怎么还原才能使这个科技树是合理的，或者告诉 CubeCat 他一开始设计的科技树就是错的，无法更正。

也就是说，给定一个由 $n$ 个顶点和 $m$ 条边组成的图。不保证给定的图是连通的。有些边已经有了方向，而其他边是无向的，你必须对无向边进行定向，使得最终得到的图是有向无环的（即所有边都有向，没有有向环的图）。

注意：你必须对所有无向边进行定向。

## 输入格式

第一行包含两个整数 $n,m$ $(2\le n\le2\times10^5,1\le m \le \min\{2\times10^5,\frac{n(n−1)}{2}\})$，分别代表节点数和边数。

接下来 $m$ 行，每行包含三个整数 $t$，$u$，$v$ $(t\in\{0,1\},1\le u,v\le n)$ ，含义如下：

- 若 $t=0$，代表该条边为无向边。

- 若 $t=1$，代表该条边为有向边，方向由 $u$ 指向 $v$ 。

## 输出格式

第一行包含一个字符串，代表最后能否修改成功。如果最后可以修改成功，输出 $\mathtt{YES}$，否则输出 $\mathtt{NO}$ 。

如果最后可以修改成功，接下来 $m$ 行，每行包含两个整数 $u,v$，表示该条边由 $u$ 指向 $v$ 。

## 输入输出样例

**输入 #1**

```
5 5
0 2 1
1 1 5
1 5 4
0 5 2
1 3 5
```

**输出 #1**

```
YES
2 1
1 5
5 4
2 5
3 5
```

**输入 #2**

```
4 5
1 4 1
1 1 3
0 1 2
1 2 4
1 3 2
```

**输出 #2**

```
NO
```
